Project l0ckcr4ck3r

(7/2022-8/2022)

L0ckcr4ck3r is a first-of-its-kind robot that was featured on Hackaday and Arduino. It uses a decoding algorithm invented by Samy Kamkar and can decode most combination locks in under 40 seconds.

Features:

  • Custom PCB
  • USB C Power Input
  • AS5600 Magnetic Encoder
  • User-based design
