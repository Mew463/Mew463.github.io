ESP32 Caliper

(1/2023-3/2023)

PCB with an ESP32 C3 which decodes the serial data output from a caliper and then sends it to a computer via bluetooth. Showcased on Hackaday.

Features:

  • Realtime Battery reporting
  • Two user buttons & 1 neopixel
  • 20 day standby time or 2hr usage
  • USB C for charging/updating firmware
