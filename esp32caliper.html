ESP32 Caliper

(1/2023-3/2023)

PCB with an embedded ESP32 C3 which decodes the data output from a caliper and then sends it to a computer via bluetooth. Featured on Hackaday and Arduino. It uses a decoding algorithm invented by Samy Kamkar and can decode most combination locks in under 40 seconds.

Features:

  • Realtime Battery reporting
  • Two user buttons
  • 20 day standby time or 2hr usage
  • USB C for charging/uploading new code
